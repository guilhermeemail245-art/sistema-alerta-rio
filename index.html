Fontes dos dados

Transparência
Nível do rio: dados medidos pelo ESP32 com sensor ultrassônico e enviados para o banco Supabase.
Meteorologia: previsão de temperatura, chuva, probabilidade de chuva e vento via API Open-Meteo.
Radar/mapa meteorológico: visualização interativa via Windy, usada para acompanhar chuva e áreas de instabilidade na região.
Análise inteligente: lógica criada com apoio de IA para interpretar nível, tendência e previsão de chuva. O projeto é experimental e não substitui alertas oficiais.
