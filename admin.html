Resumo do SMS

O SMS agora é automático: quando o sistema registra INUNDAÇÃO, a Edge Function envia ou registra o alerta.

Os botões criam registros na tabela comandos. O ESP32 consulta essa tabela e executa automaticamente quando estiver online. O envio de SMS não usa mais SIM800L: ele é feito pelo Supabase com Edge Function e API de SMS.
